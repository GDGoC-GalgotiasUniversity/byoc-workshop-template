# 🚀 BYOC: Build Your Own Copilot

Welcome to the **Build Your Own Copilot** workshop! This template is your starting point for building a personal AI assistant that can read your documents, search the web, and work in teams.

## 🛠️ Tech Stack
- **Backend:** Python (FastAPI) + LangChain + Ollama
- **Frontend:** React (Vite) + Tailwind CSS
- **Vector DB:** ChromaDB
- **Auth:** Firebase

## ⚡ Quick Start

### 1. Prerequisites
- [Ollama](https://ollama.com/) installed and running.
- [Node.js](https://nodejs.org/) (v18+) installed.
- [Python](https://www.python.org/) (3.10+) installed.

### 2. Setup & Installation
Run the setup script to create a virtual environment, install dependencies, and setup the frontend:
```bash
# Mac/Linux
bash setup.sh

# Windows (Git Bash or WSL)
bash setup.sh
```

### 3. Running the App
**Start the Backend:**
```bash
source venv/bin/activate  # Windows: venv\Scripts\activate
python -m backend.main
```

**Start the Frontend:**
```bash
cd frontend
npm run dev
```
